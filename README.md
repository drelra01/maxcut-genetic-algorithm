# Maximum-Cut Optimization System

## Overview

This project uses a hybrid Genetic Algorithm and Wisdom of Artificial Crowds approach to solve the NP-hard Maximum-Cut problem on a 100-vertex random graph. The goal was to test whether evolutionary algorithm techniques and consensus-based voting could improve cut values compared to a random baseline.

## My Contribution

I developed the Python-based optimization system and implemented the main algorithm components, including selection, crossover, mutation, fitness evaluation, and consensus voting. I also conducted 40 experimental runs and analyzed the results to compare the algorithm’s performance against the random baseline.

## Technologies Used

* Python
* Genetic Algorithms
* Artificial Intelligence
* Optimization
* Experimental analysis

## Features

* Generates or processes a graph for the Maximum-Cut problem
* Evaluates cut values using a fitness function
* Applies genetic algorithm techniques such as selection, crossover, and mutation
* Uses Wisdom of Artificial Crowds consensus voting to improve solution selection
* Runs multiple experiments to measure average performance

## Results

The system achieved an average cut value of 1,394.8 edges across 40 experimental runs, which was approximately 157 edges above the 1,237.5-edge random baseline.

## Notes

This project was created to explore how artificial intelligence and evolutionary algorithms can be applied to difficult graph optimization problems.
